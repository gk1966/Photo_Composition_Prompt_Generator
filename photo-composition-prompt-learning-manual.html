🎯 1. Introduction to Prompt Engineering for Image Composition

Learning Objectives:

  • Understand what prompt engineering is and its role in AI-assisted image editing
  • Learn the purpose and workflow of composition prompt generation
  • Recognize the difference between image generation and image composition
  • Identify real-world applications in digital media production

1.1 What is Prompt Engineering?

Definition

Prompt Engineering is the practice of crafting precise, detailed instructions (prompts) that guide AI systems to produce desired outcomes. In the context of image composition, it involves creating specifications that tell an AI tool exactly how to combine multiple images into a cohesive final result.

This tool helps you create structured, professional prompts for AI-powered image composition tools like Claude, ChatGPT with image capabilities, Adobe Firefly, or specialized composition software.

1.2 Image Composition vs. Image Generation

Aspect Image Generation Image Composition
Input Text description only Existing images + specifications
Process Creates new images from scratch Combines existing images
Output Synthetic/generated content Layered composition of real images
Control Limited control over specific details Precise control over each element
Use Case Creating new artwork, concepts Photo editing, realistic composites
This tool creates composition prompts, not generation prompts. You must have existing images to work with. The prompt tells AI how to combine YOUR images, not how to create new ones.

1.3 Real-World Applications

  • Advertising & Marketing: Product photos with custom backgrounds
  • Social Media Content: Instagram posts, stories, collages
  • E-commerce: Product presentations with lifestyle contexts
  • Photography: Artistic composites, double exposures
  • Graphic Design: Posters, promotional materials
  • Web Design: Hero images, banners

Before using this tool effectively, you should have basic knowledge of:

  • Digital image formats (JPG, PNG, TIFF)

2.1 The Layer System

Understanding Layers

A layer is an independent image element that can be manipulated separately from other elements. Layers stack on top of each other, with the bottom layer typically being the background and upper layers containing foreground elements.

Layer Properties:

  • Z-Index: The stacking order (which layer is on top)
  • Opacity: Transparency level (0-100%)
  • Blend Mode: How the layer interacts with layers below
  • Transform: Position, size, rotation
  • Effects: Blur, brightness, contrast, etc.

2.2 Canvas & Coordinate System

The Canvas

The canvas is your working area - the final image dimensions. Understanding canvas coordinates is essential for precise positioning.

Coordinate System: Digital images use a pixel-based coordinate system where (0,0) is typically the top-left corner. X increases to the right, Y increases downward.

Position Terminology:

  • Top-Left: (0%, 0%) or (0px, 0px)
  • Center: (50%, 50%)
  • Bottom-Right: (100%, 100%)

2.3 Resolution & Quality

Term Definition Typical Values
Resolution Number of pixels (width × height) 1920×1080, 3840×2160
DPI/PPI Dots/Pixels per inch (print quality) 72 (web), 300 (print)
Aspect Ratio Width to height proportion 16:9, 4:3, 1:1
Bit Depth Color information per pixel 8-bit, 16-bit, 32-bit

2.4 Color Spaces

Common Color Spaces in Digital Imaging

  • RGB (Red, Green, Blue): Standard for digital displays. Additive color model.
  • CMYK (Cyan, Magenta, Yellow, Black): Used for printing. Subtractive color model.
  • sRGB: Standardized RGB color space for web and general use.
  • Adobe RGB: Wider color gamut for professional photography.
  • HSL/HSV: Hue, Saturation, Lightness/Value - intuitive for color adjustments.
For web and social media, always use sRGB color space. For print, convert to CMYK before sending to press.

📐 4. Understanding Layers & Composition Theory

Learning Objectives:

  • Master the layer stacking paradigm
  • Apply composition rules (Rule of Thirds, Golden Ratio)
  • Understand depth and perspective in composition
  • Learn to create visual hierarchy

4.1 Layer Stacking Order

The Z-Index Concept

Layers are stacked in the order you add them:

  • Layer 1 (First added): Bottom - Usually background
  • Layer 2: Above Layer 1 - Mid-ground elements
  • Layer 3: Above Layer 2 - Foreground elements
  • Layer N (Last added): Top - Details, overlays, effects
Think of layers like sheets of glass stacked on top of each other. You look down through all the sheets to see the final composition.

4.2 Composition Principles

Rule of Thirds

Divide your canvas into a 3×3 grid. Place important elements along the lines or at intersection points.

  • Strong compositional points: (33%, 33%), (66%, 33%), (33%, 66%), (66%, 66%)
  • Creates visual balance and interest
  • Avoid centering everything - it's static

Golden Ratio (φ ≈ 1.618)

More sophisticated than Rule of Thirds, based on mathematical proportion found in nature.

  • Primary focal point: approximately 38% from edge
  • Secondary focal point: approximately 62% from edge
  • Creates naturally pleasing compositions

Visual Weight & Balance

Elements have "visual weight" based on:

  • Size: Larger = heavier
  • Color: Warm/bright colors = heavier
  • Contrast: High contrast = heavier
  • Complexity: Detailed areas = heavier

Balance heavy elements on one side with lighter elements or negative space on the other.

4.3 Depth & Perspective

Creating Depth Illusion

Make 2D compositions feel three-dimensional:

Technique How to Apply
Size Variation Larger objects = closer; smaller = farther
Overlap Objects in front partially cover those behind
Blur Gradient Sharp foreground, blurred background (depth of field)
Color Temperature Warm foreground, cool background (atmospheric perspective)
Contrast High contrast foreground, low contrast background

4.4 Layer Purpose Classification

The tool asks you to classify each layer's purpose. Understanding these categories is crucial:

Purpose Description Typical Settings
Background The backdrop; fills entire canvas 100% opacity, Size: 100%, Position: fill
Main Subject Primary focus of composition High opacity (80-100%), prominent position, masking often needed
Specific Part Supporting elements, details, overlays Variable opacity and blend modes, strategic positioning

⚙️ 5. Technical Parameters Explained

Learning Objectives:

  • Understand each adjustable parameter's function
  • Learn appropriate value ranges for different effects
  • Master the interaction between multiple parameters
  • Apply parameters strategically for desired outcomes

5.1 Transform Parameters

Position

Determines where on the canvas the layer's reference point is placed.

Options:

  • Top-Left, Top-Center, Top-Right - Upper canvas positions
  • Center-Left, Center, Center-Right - Middle row positions
  • Bottom-Left, Bottom-Center, Bottom-Right - Lower canvas positions
  • Custom - Specify exact X, Y coordinates or percentages

Example Use Cases:

✅ Portrait Photography:

Subject at Center or Rule of Thirds position (66% from left)

✅ Product Photos:

Product at Center with symmetric background

✅ Text Overlays:

Text at Top-Center or Bottom-Center for readability

Size

Controls the scale of the layer relative to the canvas.

Range: 10% to 200%

  • 10-30%: Small details, logos, decorative elements
  • 40-60%: Medium subjects, balanced composition
  • 70-100%: Large subjects, backgrounds, full coverage
  • 110-200%: Zoom-in effect, cropped view, dramatic close-ups
Scaling up beyond 100% can cause pixelation if the original image resolution is insufficient. Always use high-resolution source images for upscaling.

Rotation

Rotates the layer around its center point.

Range: -180° to +180°

  • 0°: No rotation (default)
  • ±5-15°: Subtle dynamic tilt (common in collages)
  • 90° or -90°: Portrait ↔ Landscape orientation
  • 180°: Complete inversion
Use small rotation angles (±5-10°) to add energy and break rigid layouts. Large rotations should have compositional purpose.

5.2 Opacity & Visibility

Opacity

Controls layer transparency from completely transparent (invisible) to completely opaque.

Range: 0% to 100%

Value Effect Common Use
0% Invisible Hiding layer temporarily
10-30% Very subtle Watermarks, subtle textures
40-60% Translucent Overlays, ghost effects, blending
70-90% Mostly opaque Soft integration with background
100% Fully opaque Primary subjects, backgrounds
Alpha Channel: Opacity is achieved through the alpha channel in image processing. An alpha value of 0 means fully transparent, while 255 (or 1.0 in normalized form) means fully opaque.

5.3 Color & Light Adjustments

Brightness

Adjusts the overall lightness or darkness of the layer.

Range: -100% to +100%

  • Negative values: Darken the image
  • 0%: Original brightness (default)
  • Positive values: Lighten the image
When combining multiple images, match brightness levels for coherent lighting. Background slightly darker than foreground creates depth.

Contrast

Controls the difference between light and dark areas.

Range: -100% to +100%

  • Negative values: Flatten image, reduce difference between lights/darks
  • 0%: Original contrast (default)
  • Positive values: Enhance difference between lights/darks (more punchy)

Contrast Application:

✅ Low Contrast (-30 to -50):

Soft, dreamy backgrounds; vintage filter effects

✅ High Contrast (+30 to +50):

Dramatic portraits; enhancing product details; text legibility

Saturation

Controls color intensity/vividness.

Range: -100% to +100%

  • -100%: Grayscale (no color)
  • -50%: Muted, desaturated colors
  • 0%: Original saturation (default)
  • +50%: Vivid, enhanced colors
  • +100%: Maximum color intensity (can look artificial)

5.4 Effects & Filters

Blur

Applies Gaussian blur to soften the image.

Range: 0px to 50px

Blur Amount Effect Use Case
0px Sharp (no blur) Default; primary subjects
2-5px Subtle softening Soft focus portraits; gentle depth of field
10-20px Moderate blur Background separation; de-emphasizing elements
25-50px Strong blur Abstract backgrounds; bokeh effect; privacy blurring
Gaussian Blur: The most common blur algorithm, named after the Gaussian distribution (bell curve). It produces natural-looking blur with smooth falloff.

🎨 6. Blend Modes: Theory & Application

Learning Objectives:

  • Understand the mathematical basis of blend modes
  • Learn when to use each blend mode type
  • Master creative applications of blending
  • Recognize blend mode categories and behaviors

6.1 What Are Blend Modes?

Definition

Blend modes are mathematical algorithms that determine how pixels in a layer combine with pixels in layers beneath it. They calculate the final color based on the source color (current layer) and destination color (layers below).

Formula: Result = BlendFunction(Source, Destination)

6.2 Blend Mode Categories

Normal Mode

Algorithm: Direct overlay with opacity control

Formula: Result = Source × Opacity + Destination × (1 - Opacity)

Characteristics:

  • No color interaction between layers
  • Simply places one image over another
  • Opacity is the only variable

Use when: You want straightforward layering without color effects

Darken Modes (Multiply)

Algorithm: Multiplies color values

Formula: Result = (Source × Destination) / 255

Characteristics:

  • Always produces darker colors (never lighter)
  • White (255) becomes transparent
  • Black (0) remains black
  • Mimics overlaying colored glass or filters

Use for:

  • Adding shadows
  • Texture overlays on photographs
  • Color grading with darker tones
  • Simulating traditional print methods

Practical Example:

Scenario: Adding wood texture to a photo

Setup:

  • Layer 1: Your photograph
  • Layer 2: Wood texture image
  • Blend Mode: Multiply
  • Opacity: 40-60%

Result: The wood grain appears naturally integrated, darkening the image while preserving detail.

Lighten Modes (Screen)

Algorithm: Inverted multiplication (opposite of Multiply)

Formula: Result = 255 - ((255 - Source) × (255 - Destination) / 255)

Characteristics:

  • Always produces lighter colors (never darker)
  • Black (0) becomes transparent
  • White (255) remains white
  • Mimics projecting light through layers

Use for:

  • Adding light effects (lens flares, glows)
  • Brightening specific areas
  • Creating luminous overlays
  • Simulating fog or mist

Practical Example:

Scenario: Adding sun flare to landscape

Setup:

  • Layer 1: Landscape photo
  • Layer 2: Sun flare image (bright on black background)
  • Blend Mode: Screen
  • Opacity: 60-80%

Result: The bright parts of the flare add to the scene while the black background becomes invisible.

Contrast Modes (Overlay)

Algorithm: Combination of Multiply and Screen based on destination brightness

Formula:

If Destination < 128: Result = 2 × Source × Destination / 255 If Destination ≥ 128: Result = 255 - 2 × (255 - Source) × (255 - Destination) / 255

Characteristics:

  • Darkens dark areas (Multiply)
  • Lightens light areas (Screen)
  • Increases overall contrast
  • 50% gray has no effect

Use for:

  • General purpose blending
  • Adding textures while preserving contrast
  • Color grading overlays
  • Enhancing depth

Soft Light

Algorithm: Gentler version of Overlay

Characteristics:

  • Similar to Overlay but more subtle
  • Mimics soft diffused lighting
  • Preserves more detail than Overlay

Use for:

  • Subtle color corrections
  • Gentle texture overlays
  • Portrait retouching
  • Creating mood without over-processing

Hard Light

Algorithm: More intense version of Overlay

Characteristics:

  • Stronger effect than Overlay or Soft Light
  • Simulates harsh lighting conditions
  • Can create dramatic contrast

Use for:

  • Dramatic lighting effects
  • High-contrast artistic looks
  • Strong texture applications
  • Creating impact

6.3 Blend Mode Selection Guide

Goal Recommended Blend Mode Opacity Range
Basic layering Normal 80-100%
Add texture Multiply or Overlay 30-60%
Add light/glow Screen 40-70%
Color grading Soft Light or Overlay 20-50%
Dramatic effect Hard Light 50-80%
Subtle integration Soft Light 20-40%
Blend modes affect color and tone relationships. If your composition looks "off" after applying a blend mode, check if all layers have consistent white balance and color temperature.

✂️ 7. Masking & Edge Control Techniques

Learning Objectives:

  • Understand different masking techniques
  • Master edge feathering for natural blending
  • Learn background removal strategies
  • Apply selective visibility controls

7.1 What is Masking?

Definition

Masking is a non-destructive technique that controls which parts of a layer are visible and which are hidden, without permanently deleting pixels.

Mask Types:

  • Binary Mask: Pixels are either fully visible or fully hidden
  • Gradient Mask: Smooth transition from visible to hidden
  • Alpha Mask: Uses grayscale values to control transparency

7.2 Background Removal

Subject Isolation

The most common use of masking is removing backgrounds to isolate subjects.

Mask Types in the Tool:

  • None: No masking (full image visible)
  • Remove Background: Automatically isolate the main subject
  • Keep Center, Fade Edges: Vignette effect
  • Custom Shape: Geometric or custom boundary

Typical Workflow:

Portrait with New Background:

  1. Layer 1: New background scene (landscape, studio, etc.)
  2. Layer 2: Portrait photo with "Remove Background" mask
  3. Result: Subject cleanly placed in new environment

7.3 Edge Feathering

Understanding Feathering

Edge feathering creates a gradual transition at the edges of a layer, blending it smoothly with underlying layers.

Range: 0px to 200px

Feather Amount Effect Best For
0px Hard edges Geometric shapes, text, clean cuts
10-20px Slight softening Natural photos with good isolation
30-50px Noticeable blend Portraits, organic subjects
75-100px Soft integration Dreamy effects, subtle overlays
150-200px Strong fade Vignettes, transitional elements
When compositing objects with fine details (hair, fur, foliage), use moderate feathering (15-30px) to avoid the "cut-out" look while preserving detail.

7.4 Advanced Masking Techniques

Gradient Masks for Depth

Create realistic depth of field by varying opacity across the image:

  • Foreground: Sharp, high opacity
  • Mid-ground: Medium opacity
  • Background: Lower opacity or more blur

Edge Refinement Strategies

For challenging edges (like hair or fur):

  1. Start with subject isolation mask
  2. Add edge feathering (20-40px)
  3. Slightly reduce opacity (95-98%) for natural integration
  4. Match lighting direction and intensity with background

7.5 Shadow Effects

Adding Realistic Shadows

Shadows ground objects in their environment. The tool includes shadow options:

Parameters:

  • Shadow Offset: How far the shadow is from the object
  • Shadow Blur: Softness of the shadow
  • Shadow Color: Usually black or dark version of background color
  • Shadow Opacity: Strength of the shadow

Shadow Guidelines:

  • Direct sunlight: Hard shadows, low blur (5-10px)
  • Overcast/diffused: Soft shadows, high blur (20-40px)
  • Indoor artificial: Medium shadows, moderate blur (10-20px)

🎨 8. Color Theory & Image Adjustments

Learning Objectives:

  • Apply color harmony principles to compositions
  • Understand color temperature and mood
  • Master color correction for consistent compositions
  • Learn about white balance and color casts

8.1 Color Harmony in Composition

Complementary Colors

Colors opposite on the color wheel create high contrast and visual interest:

  • Blue ↔ Orange
  • Red ↔ Green (use carefully - can look like Christmas)
  • Purple ↔ Yellow

Use for: Creating vibrant, energetic compositions

Analogous Colors

Colors next to each other on the color wheel create harmony:

  • Blue → Blue-green → Green
  • Red → Red-orange → Orange

Use for: Calm, cohesive, naturally pleasing compositions

8.2 Color Temperature

Understanding Temperature

Color temperature describes the warmth or coolness of an image:

Temperature Colors Mood/Association
Warm Red, Orange, Yellow Energy, passion, comfort, warmth
Cool Blue, Cyan, Purple Calm, professional, sadness, distance
Neutral Green, Gray, Beige Balance, nature, subtlety
When compositing, all layers should have consistent color temperature. A warm subject on a cool background (or vice versa) looks unnatural unless you're creating a specific artistic effect.

8.3 Color Correction Workflow

Matching Color Between Layers

To create cohesive compositions, adjust color properties:

  1. White Balance: Ensure all layers have the same color temperature
  2. Saturation: Match color intensity across layers
  3. Hue Shift: If needed, shift hues to match (e.g., make all layers slightly warmer)
  4. Brightness: Adjust so lighting looks consistent
  5. Contrast: Match contrast levels for uniform processing

8.4 Saturation Strategy

When to Adjust Saturation

Increase Saturation (+10 to +40%):

  • Dull or flat-looking images
  • Product photography (make colors pop)
  • Social media content (draws attention)
  • Travel/landscape photos

Decrease Saturation (-10 to -40%):

  • Over-saturated images
  • Matching different image sources
  • Creating muted, sophisticated looks
  • Portrait backgrounds (to emphasize subject)

Remove Saturation (-100%, Grayscale):

  • Black and white artistic effect
  • Reducing background distraction
  • Vintage/classic look
Over-saturation (>+50%) can make images look artificial and hurt color fidelity. Use moderation unless seeking a specific hyper-colorful aesthetic.

9.3 File Format Decision Matrix

Format Comparison

Format Compression Transparency Best For
JPG/JPEG Lossy No Photographs, web, small file size, social media
PNG Lossless Yes (Alpha) Graphics, logos, images needing transparency, web
TIFF None or lossless Yes Print, archival, professional photography
WebP Lossy or lossless Yes Modern web, Google platforms, smaller than PNG/JPG

Format Selection Examples:

✅ Instagram Post:

JPG, 1080×1080, quality 85-95%

✅ Website Hero Image:

WebP (with JPG fallback), 1920×1080, quality 80%

✅ Logo on Website:

PNG with transparency, appropriate size (2-4× display size for retina)

✅ Print Magazine:

TIFF, 300 DPI, CMYK color space

9.4 Quality vs. File Size

Optimization Strategies

For Web (Priority: Fast Loading):

  • Use JPG quality 75-85% for photographs
  • Use PNG-8 for simple graphics (256 colors)
  • Consider WebP for modern browsers
  • Resize to exact display dimensions (no larger)

For Social Media (Priority: Quick Upload, Decent Quality):

  • Follow platform recommendations (Instagram: 1080×1080)
  • JPG quality 85-95%
  • sRGB color space

For Print (Priority: Maximum Quality):

  • 300 DPI minimum
  • TIFF or high-quality JPG (95-100%)
  • CMYK color space
  • Include bleed area (extra margin)

For Portfolio/Archive (Priority: Longevity):

  • PNG or TIFF
  • High resolution
  • No compression or lossless

9.5 Advanced Output Settings

Additional Options Explained

HDR Output:

  • High Dynamic Range - greater range of brightness
  • Use for: Modern displays, HDR-compatible content
  • Requires: HDR source images or tone mapping

sRGB Color Profile:

  • Standard color space for web and general viewing
  • Use for: Almost everything digital
  • Ensures consistent color across different displays

Preserve Details:

  • Sharpening and detail enhancement during resize
  • Use for: Downscaling images, preventing softness
  • Can add slight grain at high settings

Anti-Aliasing:

  • Smooths jagged edges on diagonal lines and curves
  • Use for: Text, graphics, geometric shapes
  • Usually enabled by default

🎓 10. Practical Exercises & Projects

Learning Objectives:

  • Apply learned concepts to real projects
  • Build practical prompt engineering skills
  • Develop a personal workflow
  • Create portfolio-worthy compositions

Exercise 1: Simple Background Replacement

Beginner

Objective: Replace the background of a portrait with a new scene.

Requirements:

  • 1 portrait photo (person against simple background)
  • 1 background image (landscape, studio, etc.)

Steps:

  1. Add background image as Layer 1
    • Position: Center, Size: 100%
    • Apply slight blur (5-10px) for depth
  2. Add portrait as Layer 2
    • Description: Describe the person and their pose
    • Purpose: Main subject
    • Mask Type: Remove background
    • Edge Feathering: 20-30px
  3. Match colors
    • Adjust brightness and saturation to match background
    • Ensure color temperature is consistent
  4. Add shadow (optional but recommended)
  5. Set output: 4:3, High resolution, PNG

Success Criteria:

  • Clean edge separation between subject and background
  • Natural-looking integration (no "cut-out" appearance)
  • Consistent lighting direction

Exercise 2: Instagram Square Collage

Beginner

Objective: Create a 3-photo collage for Instagram.

Requirements:

  • 3 related photos (vacation, portfolio samples, event photos)
  • 1 background (solid color or subtle texture)

Steps:

  1. Layer 1: Background (white, cream, or subtle pattern)
  2. Layer 2: Photo 1 (left side)
    • Position: Custom - 25% from left, centered vertically
    • Size: 40%
    • Rotation: -5° for dynamic feel
    • Shadow: Enabled
  3. Layer 3: Photo 2 (center)
    • Position: Center
    • Size: 45%
    • No rotation (or minimal)
    • Shadow: Enabled
  4. Layer 4: Photo 3 (right side)
    • Position: Custom - 75% from left, centered vertically
    • Size: 40%
    • Rotation: +5°
    • Shadow: Enabled
  5. Output: 1:1 (square), 1080×1080, JPG

Exercise 3: Product Photography Composite

Intermediate

Objective: Place a product in a lifestyle scene.

Requirements:

  • 1 product photo (on white/transparent background)
  • 1 lifestyle scene (kitchen counter, desk, outdoor table)
  • Optional: 1 texture overlay

Steps:

  1. Layer 1: Lifestyle scene background
  2. Layer 2: Product
    • Remove background
    • Position strategically using Rule of Thirds
    • Size: 30-50% depending on composition
    • Edge feathering: 15-25px
    • Shadow: Yes - match lighting of scene
  3. Layer 3 (optional): Subtle texture overlay
    • Blend mode: Multiply or Soft Light
    • Opacity: 15-30%
  4. Color corrections
    • Match product white balance to scene
    • Adjust brightness for lighting consistency
  5. Output: 16:9 for web, High resolution

Challenge: Make the lighting look 100% natural.

Exercise 4: Double Exposure Effect

Intermediate

Objective: Create an artistic double exposure.

Requirements:

  • 1 portrait (profile or silhouette works best)
  • 1 texture/landscape (nature, cityscape, abstract)

Steps:

  1. Layer 1: Portrait
    • High contrast (+30-50%)
    • Potentially desaturate slightly
  2. Layer 2: Texture/landscape
    • Blend mode: Screen or Overlay
    • Opacity: 50-80%
    • Position to align interesting elements with face
  3. Fine-tune
    • Adjust saturation for artistic effect
    • May need to brighten or darken individual layers
  4. Output: 4:3 or 1:1, High resolution, PNG

Exercise 5: Multi-Layer Complex Composition

Advanced

Objective: Create a poster-quality composition with 5+ layers.

Requirements:

  • 1 background scene
  • 2-3 main subjects (people, products, or elements)
  • 1-2 texture/overlay layers
  • 1 text/graphic layer (represented by description)

Steps:

  1. Plan your composition on paper first
  2. Layer 1: Background with atmospheric blur
  3. Layer 2-4: Main subjects
    • Vary sizes for depth perception
    • Use foreground/mid-ground/background positioning
    • Apply different blur levels
  4. Layer 5: Color grading overlay
    • Solid color with Soft Light or Overlay blend
    • Low opacity (15-30%)
  5. Layer 6: Texture or light effect
    • Appropriate blend mode
    • Strategic opacity
  6. Layer 7: Text/graphic space (describe placement)
  7. Output: Custom dimensions, Ultra resolution, PNG or TIFF

Evaluation Criteria:

  • Visual hierarchy (clear focus)
  • Color harmony
  • Lighting consistency
  • Professional finish quality
  • Creative use of blend modes
After completing each exercise, save your prompt file and document what worked well and what you'd change. Building a personal prompt library accelerates future work.

🚀 11. Advanced Techniques

Learning Objectives:

  • Master advanced composition strategies
  • Learn prompt optimization techniques
  • Understand iterative refinement workflow
  • Develop professional-level skills

11.1 Advanced Layer Strategies

Adjustment Layers Concept

While this tool doesn't have traditional adjustment layers, you can simulate them:

  • Add a solid color layer with Overlay blend mode for color grading
  • Add a gradient layer with Soft Light for vignetting
  • Use texture overlays with Multiply for film grain

Layer Groups & Organization

When working with many layers, mentally group them:

  • Base Group: Background and environment (Layers 1-2)
  • Subject Group: Main focus elements (Layers 3-5)
  • Effects Group: Overlays, textures, color grading (Layers 6-8)
  • Detail Group: Fine details, highlights, shadows (Layers 9+)

11.2 Prompt Optimization

Writing Better Descriptions

The quality of your layer descriptions directly affects results:

Description Quality Comparison:

❌ Poor Description:

"A person"

⚠️ Okay Description:

"Woman in red dress"

✅ Good Description:

"Professional portrait of woman, age 30, wearing red cocktail dress, neutral studio lighting"

✅✅ Excellent Description:

"Professional portrait: Caucasian woman, early 30s, long brunette hair, wearing red cocktail dress, standing pose, soft studio lighting from camera-left at 45°, neutral gray background, shot with 85mm lens creating shallow depth of field"

Key Elements to Include:

  • Subject type and details
  • Colors and materials
  • Lighting characteristics
  • Perspective or angle
  • Style or mood
  • Technical details if relevant

11.3 Lighting Coherence

Matching Light Direction

For realistic compositions, light must be consistent:

  1. Identify light direction in your background (where are highlights/shadows?)
  2. Ensure all subjects have lighting from the same direction
  3. Match light intensity (brightness) across layers
  4. Match light color temperature (warm/cool)
  5. Add shadows that match the light source
Professional Tip: When compositing outdoors, sun position is key. Subjects should all have sun highlights on the same side. Indoor scenes should have consistent window or lamp direction.

11.4 Iterative Refinement

The Professional Workflow

  1. Draft Pass:
    • Rough layout with basic positioning
    • Generate prompt and test
    • Evaluate overall composition
  2. Refinement Pass:
    • Adjust problematic areas
    • Fine-tune opacity and blend modes
    • Add edge feathering
    • Regenerate and test
  3. Detail Pass:
    • Perfect color matching
    • Add shadows and highlights
    • Apply final effects
  4. Polish Pass:
    • Overall color grading
    • Sharpening/detail enhancement
    • Final quality check

11.5 Common Problems & Solutions

Problem Cause Solution
Subjects look "cut out" Hard edges, no feathering Add 20-40px edge feathering, reduce opacity slightly (98%)
Floating objects No shadows Add shadows matching light direction and intensity
Color mismatch Different white balance Adjust saturation, brightness, potentially add color overlay
Inconsistent lighting Sources lit from different directions Adjust brightness per layer, add directional highlights
Scale looks wrong Size proportions unrealistic Research real-world proportions, adjust sizes accordingly
Flat/no depth Everything same sharpness/contrast Apply blur gradient, vary contrast, adjust saturation by depth

Knowledge Check Questions

Question 1: Layer Order

You're creating a composition with a landscape background, a person, and a text overlay. What's the correct layer order from bottom to top?

A) Text, Person, Landscape
B) Person, Landscape, Text
C) Landscape, Person, Text
D) It doesn't matter

Correct Answer: C - Background first, then subject, then overlay elements on top.

Question 2: Blend Modes

You want to add a wood texture to a photo but keep the photo details visible while darkening the overall tone. Which blend mode should you use?

A) Normal
B) Screen
C) Multiply
D) Overlay

Correct Answer: C - Multiply darkens while preserving detail and making white areas transparent.

Question 3: Opacity

What opacity range is most appropriate for a subtle watermark?

A) 10-30%
B) 50-70%
C) 80-90%
D) 100%

Correct Answer: A - Watermarks should be visible but not distracting.

Question 4: Resolution

You're creating an image for a printed 8×10" photo. What minimum DPI should you use?

A) 72 DPI
B) 150 DPI
C) 300 DPI
D) 600 DPI

Correct Answer: C - 300 DPI is the professional standard for photo printing.

Question 5: Edge Feathering

You're compositing a portrait onto a new background. The edges look too sharp and artificial. What's the best solution?

A) Reduce opacity to 50%
B) Add 20-40px edge feathering
C) Change blend mode to Multiply
D) Apply maximum blur

Correct Answer: B - Edge feathering creates natural transitions without affecting the overall image.

Question 6: File Formats

Which format should you use for a logo that needs a transparent background for web use?

A) JPG
B) PNG
C) TIFF
D) WebP

Correct Answer: B - PNG supports transparency and is widely supported on the web.

Question 7: Color Temperature

Your subject was photographed under warm indoor lighting, but your background is a cool outdoor scene. What should you do?

A) Leave both as-is for contrast
B) Make the background warmer or the subject cooler
C) Convert everything to black and white
D) Use Screen blend mode

Correct Answer: B - Matching color temperature is essential for realistic compositions.

Question 8: Composition

According to the Rule of Thirds, where should you position your main subject for the strongest composition?

A) Dead center (50%, 50%)
B) At intersection points (~33% or 66% from edges)
C) In a corner
D) Randomly positioned

Correct Answer: B - Rule of Thirds intersection points create dynamic, balanced compositions.
